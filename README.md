# 🎲 席替えアプリ

高校のクラスで使うための「席替えWebアプリ」です。  
楽しい演出で席替えを盛り上げながら、座席表作成の手間を省きます。

---

## ✅ 実装済み機能

### STEP 1｜生徒登録
- **一括コピー＆ペースト入力**（タブ区切り・カンマ・スペース対応）
- **英語名対応**：`1 鈴木朋子 Tomoko Suzuki` 形式で英語名も一緒に入力可
  - 英語名は任意（なくてもOK）
- 個別追加（番号・日本語名・英語名フィールドあり）
- 出席番号順ソート / 参加・欠席切り替え / 削除
- サンプルデータ読み込み（36名・英語名つき）
- クラス名・実施日の設定

### STEP 2｜座席配置
- 縦・横の列数を自由設定（最大12×12）
- 教卓を前方に固定表示
- 机をクリックして空席 ON/OFF
- 空席リセット・自動設定
- 🎨 **座席カラー設定**
  - 枠色・背景色・文字色をカラーピッカーで自由変更
  - 9種のプリセット（青・緑・オレンジ・赤・パープル・ティール・ダーク・黄・ピンク）
  - デフォルトに戻すボタン

### STEP 3｜席替え抽選
- 暗転＋星空背景のプロジェクター映えする演出
- 3〜5秒間の高速ロールアニメーション
- 1席ずつ緑色で順次確定するアニメーション
- 確定時に紙吹雪（confetti）＋フラッシュ演出
- 🔊 **効果音（Web Audio API）**
  - ロール中：ランダムな電子音でドキドキ感
  - 確定時：音程が上がるチャイム音
  - 完了時：ファンファーレ
  - スタート時：ドン！という衝撃音
  - 効果音 ON/OFF スイッチ・音量スライダーあり
  - オーバーレイ中も音のON/OFF切替可能

### STEP 4｜結果・出力
- 座席表ビュー（教室レイアウト・教卓・前後左右表示）
- 🌏 **表示モード切替**：日本語 / 英語 / 両方
  - 英語名未登録の生徒は日本語で表示
- 名簿ビュー（番号順・英語名列あり）
- クラス名・日付・タイトル編集
- **Excel (.xlsx) ダウンロード**
  - Sheet1：日本語座席表
  - Sheet2：英語座席表（英語名登録がある場合のみ）
  - Sheet3：名簿（番号・日本語名・英語名・座席位置）
- 印刷機能（印刷用CSS対応）
- 「もう一度」で再抽選

## 📋 操作手順

1. **STEP 1**：生徒を登録（一括ペーストが便利。英語名も入力可）
2. **STEP 2**：列数設定・空席指定・座席の色を好みに変更
3. **STEP 3**：効果音を確認して「席替えスタート！」
4. **STEP 4**：日本語/英語切り替え後にExcelダウンロード
